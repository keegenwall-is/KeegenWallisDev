IoT Bay

IoT Bay is an online store that sells an assortment of different products. Users are able to browse and purchase items as well as and after creating an account can save items to a wishlist and save and update their payment and user information.

Development Process

IoT Bay was made as apart of the Introduction to Software Deveopment Subject and so only required each member to be responsible for the creation of two feature, one worked on as a team released in R0 and another one developed using their own MVC, release in R1 and touched up for R2.

View GitHub Repo

Challenges in Development

This was my first time creating an enterprise web system and as a result, a lot needed to be learned:

Deviation from the Path: The lectures and notes for this subject were all done with Java, however, because most of the team had more experience in C#, we decided that using C# would be a more comfortable direction for the team to move in. This did mean however, that most of the process of IoTBay's creation had to be self-taught, as we were using a different IDE and language than the course taught.

Data Validation: When registering a new user there are checks that need to be passed to ensure that a the user does not already have an account and that they have entered a valid password.
